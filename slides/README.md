slides
=====

Scalable HTTP Erlang presetation

Build
-----


    $ make build

Release
-----

    $ make release


Run one node
-----


    $ make run-one

Page:  http://localhost:8080


Join nodes  to the node 1
-----


    $ make cluster

```
Page:  http://localhost:8081

```
